Sign-off round — the seven new screens

Same locked system (graphite night, quiet serif, light numerals, ink buttons) applied to the rest of the MVP. The full 14-screen app is now clickable.

  • Onboarding (5 steps): tap "Play onboarding flow" above. Four questions, then the Apple Health ask framed as coach value. No account wall — per the blueprint, sign-in comes after the first workout.
  • Paywall: soft and skippable, annual pre-selected, free tier explicitly protected in the fine print. Ink, no urgency theatrics.
  • Program: from Today → "View program". Week chips, day list, today's session tappable.
  • Library → Exercise detail: from Program. Tap Bench Press for the detail page (demo video slot, light-numeral stats, history with a coral PR tag).
  • Progress photos: Progress tab → "Body" segment. Privacy promise leads the screen.
  • Quick-log sheet: tap the Body weight tile on Today.
Sign-off checklist
  1. Run onboarding in both modes
  2. Dismiss + revisit paywall (Profile → Manage subscription)
  3. Today → Program → Library → Bench Press
  4. Progress → Body → photos
  5. Log a weight from Today
