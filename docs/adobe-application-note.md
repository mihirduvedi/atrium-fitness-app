# Adobe Application Note

This project is useful for my Adobe Student Summit application because it shows the type of work I want to do more of: combining software engineering, product design, fitness, and creative storytelling.

Atrium is not just a code project. It needs:

- a clear visual identity
- polished onboarding graphics
- app store screenshots
- product demo videos
- launch content
- social templates
- user education materials
- a design system that can scale with the product

Adobe tools could support that full creative layer. Premiere Pro could help create launch videos, demos, and short-form product content. Photoshop and Illustrator could support app store visuals, brand assets, icon exploration, and marketing graphics. Adobe Express could help create fast social and launch assets. Firefly could help with early visual exploration, moodboarding, and creative iteration.

The project gives me a concrete perspective to bring to Adobe: many student builders can code functional products, but need better ways to design, explain, and launch them. I am interested in how Adobe can help students turn technical prototypes into complete product experiences.
